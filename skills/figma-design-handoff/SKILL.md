# Figma Design Handoff

## When to Use
- Preparing designs for developer implementation
- Creating handoff documentation
- Final review before development begins

## Checklist
1. All layers named per convention (run figma-layer-organization)
2. All tokens documented (run figma-variable-management audit)
3. Components audited (run figma-component-audit)
4. Specs extracted for each screen (run figma-spec-extraction)
5. Assets identified and listed
6. States and variants documented
7. Responsive behaviour noted
