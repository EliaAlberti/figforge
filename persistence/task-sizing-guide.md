# Task Sizing for Figma Work

All tasks MUST be 10-15 minutes. Here are sizing examples:

## Right-Sized

| Task | ~Time |
|------|-------|
| Extract specs for one screen | 12 min |
| Rename layers in one frame | 10 min |
| Create one component with 3 variants | 15 min |
| Audit variables in one collection | 10 min |
| Create one FigJam flowchart | 12 min |

## Too Large — Split These

| Task | Split Into |
|------|-----------|
| "Organise entire file" | One task per page/section |
| "Build full component library" | One task per component |
| "Extract all specs" | One task per screen |
| "Audit design system" | Tokens audit, Component audit, Naming audit |

## Verification

Before creating a task:
- [ ] 10-15 minutes of work
- [ ] Clear completion criteria
- [ ] Fits in one git commit
- [ ] No "and" in the title (split if so)
